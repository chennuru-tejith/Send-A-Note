# LinkedIn AI Smart Writer

This repository contains the specifications and automated prompt for building the **LinkedIn AI Smart Writer** Chrome Extension.

## 🚀 Project Overview
**LinkedIn AI Smart Writer** is a production-ready Chrome Extension (Manifest V3) designed to integrate naturally into LinkedIn. It provides AI-generated, personalized networking messages directly inside the LinkedIn platform (modal connection notes and messaging chats) without requiring copy-pasting or opening external websites.

## 💡 How to Build the Extension using Antigravity
To initiate development, copy the contents of `PROMPT.md` or instruct Antigravity directly to read `PROMPT.md` and execute the steps sequentially:

1. **Scaffold the project** (Vite + React + TS + Tailwind).
2. **Build the Profile Parser** and observer services.
3. **Implement UI components** injected via Shadow DOM.
4. **Configure background worker** and integrate OpenAI API.
5. **Implement Chrome storage** for local settings.
6. **Verify, build, and test** the extension.
